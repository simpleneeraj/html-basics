## Programming Symbols

I know when i started learning programming i don't know these symbol names, so i try to give you a little notes for see when you stuck in programming.

## Popular

| Symbol | Name           |
| ------ | -------------- |
| `:`    | Colon          |
| `;`    | Semicolon      |
| `' '`  | Single Quotes  |
| `" "`  | Double Quotes  |
| `{ }`  | Curly Braces   |
| `( )`  | Round Bracket  |
| `[ ]`  | Square Bracket |

<!-- `⇆` Parentheses -->

## Math and Calculation

| Symbol | Name           |
| ------ | -------------- |
| `+`    | Addition       |
| `-`    | Subtraction    |
| `*`    | Multiplication |
| `/`    | Division       |
| `%`    | Modulus        |

## Assignment and Equal

| Symbol | Name                |
| ------ | ------------------- |
| `=`    | Equal               |
| `==`   | Is Equal To         |
| `===`  | Strict Equal To     |
| `!=`   | Is Not Equal To     |
| `!==`  | Strict Not Equal To |

## Logical

| Symbol | Name |
| ------ | ---- |
| `&&`   | AND  |
| `॥`    | OR   |
| `!`    | Not  |
